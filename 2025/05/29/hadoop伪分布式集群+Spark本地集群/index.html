    hadoop伪分布式集群+Spark本地集群

    发表于 2025-05-29 分类于 spark

    hadoop伪分布式集群部署

    sudo vim /etc/profile 配置系统环境变量

    • source /etc/profile 使用配置生效

    • cd /opt & sudo mkdir hadoop 创建安装目录

    • sudo chown -R hadoop:hadoop hadoop 目录拥有者

    • tar -zxvf hadoop-3.3.6.tar.gz 解压缩

    • vim hadoop-env.sh 配置hadoop环境变量

    vim core-site.xml

    vim hdfs-site.xml

    vim mapred-site.xml

    vim yarn-site.xml

    • hdfs namenode -format

    • start-dfs.sh/stop-dfs.sh

    • start-yarn.sh/stop-yarn.sh

    • mapred –daemon start historyserver

    Spark本地集群安装部署

    • sudo mkdir spark

    • sudo chown -R hadoop:hadoop spark

    • tar -zxvf spark-3.5.3-bin-hadoop3.tgz

    • cd ${SPARK_HOME}\conf

    vim spark-default.conf

    vim spark-env.sh

    vim workders

    cd ${SPARK_HOME}/sbin

    ./start-master.sh

    ./start-worker.sh
